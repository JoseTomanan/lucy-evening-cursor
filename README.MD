# Lucy Evening Cursor

A soft, warm dark theme for [Cursor](https://cursor.com) based on the [lucy-evening](https://github.com/Hermitter/oh-lucy-vscode-theme) color palette, with Cursor-optimized UI styling.

## Features

- Warm evening-shifted colors derived from the original Lucy palette
- Cursor-style translucent UI: softened git decorations, diagnostics, minimap highlights, bracket guides, and status bar
- Clean, low-contrast workbench that keeps focus on the code

## Installation

1. Clone or download this repository
2. Run `npm install && npm run build`
3. Copy the entire folder to your Cursor extensions directory

## Customization

Edit the colors in `src/lucy/colors.mjs`, then run `npm run build` to regenerate the theme into `dist/`.
